#Importing a Project

Importing a project allows you to import a .kProj file from your computer into Kora. Importing a project allows you to use a previously created project in a new or different way without having to recreate the forms and its fields. 

Importing a project will bring in the existing project structure, including all forms and fields. Records will not be imported with the project. 

Follow the steps below to import a project. 

1. From anywhere in Kora, click on the **Projects** option in the top menu bar. This will open a dropdown menu. <p align="center"> <img src="../projects-img/importing_a_project_1_annotated.png" width="100%" style="align:center" title="Step 1"> </p>

2. In the dropdown menu, click the **Import Project Setup** option. This will bring you to the *Import Project Setup* page. <p align="center"> <img src="../projects-img/importing_a_project_2_annotated.png" width="100%" style="align:center" title="Step 2"> </p>
    
3. On the *Import Project Setup* page, either drag and drop your project file from you computer into the box or choose the **Select the Project File here** option to choose it through a file browser. <p align="center"> <img src="../projects-img/importing_a_project_3_annotated.png" width="100%" style="align:center" title="Step 3"> </p>

4. Click the **Upload Project File** button. <p align="center"> <img src="../projects-img/importing_a_project_4_annotated.png" width="100%" style="align:center" title="Step 4"> </p>

5. Under the *Project Information* section on the *Import Project Setup* page, you have the option to enter a project name and description, or you can leave these options blank to use the name and description from the original project. <p align="center"> <img src="../projects-img/importing_a_project_5_annotated.png" width="100%" style="align:center" title="Step 5"> </p>


6. Press the **Import Project & Information** button. <p align="center"> <img src="../projects-img/importing_a_project_6_annotated.png" width="100%" style="align:center" title="Step 6"> </p>



This will automatically upload the project into your Kora instillation. 



*Note*: If you wish to learn how to export a project, visit the [Exporting a Project Guide](../projects/exporting_a_project.md).
